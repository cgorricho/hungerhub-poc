# 🤖 Gemini CLI Review Template

**FOR GEMINI CLI USE**: When reviewing Agent Mode progress, please save your assessment using this template in the `gemini_reviews/` folder.

## 📝 Review Template Format:

```markdown
# 🔍 Gemini CLI Assessment - [Component/Feature Name]

**Date**: YYYY-MM-DD  
**Reviewer**: Gemini CLI  
**Review Type**: [Code Review / Methodology Validation / Architecture Assessment]  
**Agent Mode Report**: [Reference to reviewed report]  

## 📋 Review Summary
[Brief overview of what was reviewed]

## ✅ Strengths Identified
- [List positive aspects]
- [Technical quality points]
- [Best practices followed]

## ⚠️ Issues Found
- [List concerns or problems]
- [Potential risks identified]
- [Standards violations]

## 🔧 Recommendations
- [Specific improvement suggestions]
- [Alternative approaches]
- [Best practices to implement]

## 📊 Quality Assessment
- **Code Quality**: [Rating/Comments]
- **Documentation**: [Rating/Comments]  
- **Architecture**: [Rating/Comments]
- **Testing**: [Rating/Comments]

## ✅ Approval Status
- [ ] ✅ APPROVED - Ready for next phase
- [ ] ⚠️ CONDITIONAL - Address issues before proceeding  
- [ ] ❌ NEEDS REWORK - Significant changes required

## 📝 Next Steps
[Specific actions for Agent Mode to take]

---
**Review Complete**: [Timestamp]
```

## 🎯 Usage Instructions for Gemini:
1. Copy this template
2. Fill in your assessment details
3. Save as: `gemini_reviews/YYYY-MM-DD_review_[component_name].md`
4. Reference in collaboration logs
