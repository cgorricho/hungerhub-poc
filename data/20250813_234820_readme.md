# Data Directory

## 📁 Data Storage Structure

### **raw/**
Original data extracted from Oracle database.
- Raw CSV/JSON exports
- Original Excel files (samples)
- Unprocessed data from source systems

### **processed/**
Cleaned and transformed data ready for analysis.
- Parquet files for fast I/O
- Fact and dimension tables
- Validated and deduplicated data

### **output/**
Analysis results and generated reports.
- Statistical summaries
- Generated insights
- Export files for stakeholders

## 🔄 Data Flow
```
Oracle DB → raw/ → ETL Pipeline → processed/ → Analytics → output/
```

## 📊 Expected Data Volume
- **Raw Data**: ~54K records (1% Georgia sample)
- **Processed**: Optimized Parquet format
- **Output**: JSON/CSV summary files

## 🚨 Important Notes
- Raw data files ignored in git (.gitignore)
- Use environment variables for sensitive data
- Implement data quality checks at each stage
