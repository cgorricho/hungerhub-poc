# HungerHub Dashboard Usage Guide

## 🚀 **Launch Individual Dashboards**

### **Dash Dashboard (Plotly)**
```bash
./launch_enhanced_dash.sh
```
- **Port**: 8050
- **URL**: http://localhost:8050
- **Features**: Interactive charts, multi-tab interface, real-time filtering

### **Streamlit Dashboard**
```bash
./launch_enhanced_streamlit.sh
```
- **Port**: 8503
- **URL**: http://localhost:8503  
- **Features**: Sidebar filters, metrics, data tables

## 🛑 **Stop Dashboards**

### **Stop Dash**
```bash
pkill -f "python.*enhanced_app.py"
```

### **Stop Streamlit**
```bash
pkill -f "streamlit.*enhanced_app.py"
```

### **Check Running Processes**
```bash
# Check Dash
pgrep -f "python.*enhanced_app.py"

# Check Streamlit  
pgrep -f "streamlit.*enhanced_app.py"
```

## 🔗 **VS Code Port Forwarding Setup**

1. **Open Command Palette**: `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
2. **Type**: "Forward a Port" 
3. **Enter Port**: `8050` (Dash) or `8503` (Streamlit)
4. **Access**: http://localhost:PORT in your local browser

## 🔧 **Troubleshooting**

### **If Dashboard Won't Start**
```bash
# Kill any existing processes
pkill -f "python.*enhanced_app.py"
pkill -f "streamlit.*enhanced_app.py"

# Wait a moment
sleep 2

# Try launching again
./launch_enhanced_dash.sh    # or streamlit version
```

### **Check Logs**
```bash
# Dash logs
ls -la logs/dash_app_*.log
tail -f logs/dash_app_*.log

# Streamlit logs  
ls -la logs/streamlit_app_*.log
tail -f logs/streamlit_app_*.log
```

### **Test Dependencies**
```bash
python -c "import dash, streamlit, pandas, plotly; print('✅ All dependencies OK')"
```
