# Deprecated Files

This folder contains files that are no longer actively used in the HungerHub POC project but are preserved for reference and potential future use.

## 📁 Directory Structure

```
src/deprecated/
├── analytics/           # Old analytics engine versions
├── data_extraction/     # Superseded data extraction scripts
├── dashboard/          # Broken or obsolete dashboard files
├── notebooks/          # Old analysis notebooks
├── pipeline/           # Legacy ETL pipeline implementations
└── scripts/            # Test scripts and diagnostic utilities
```

## 🗂️ Deprecated Files and Reasons

### Analytics (`analytics/`)
- **`analytics_engine.py`** - Original analytics engine with bugs
  - *Issues*: Had syntax errors and incomplete data loading logic
  - *Superseded by*: `analytics_engine_fixed.py` and new analytics in active codebase

- **`analytics_engine_fixed.py`** - Fixed version but not used in current pipeline
  - *Reason*: Works but relies on mock/synthetic data structure that doesn't match real Oracle data
  - *Status*: May be useful for reference or testing with mock data

### Data Extraction (`data_extraction/`)
- **`real_data_extractor.py`** - Original Oracle data extractor
  - *Issues*: Limited to sample-size extractions, not production-scale
  - *Superseded by*: `full_data_extractor.py` for high-volume production data extraction
  - *Note*: Still functional for small-scale testing but not recommended for production

### Dashboard (`dashboard/`)
- **`app_broken.py`** - Plotly Dash dashboard with critical issues
  - *Issues*: Hardcoded file paths, broken data loading, incomplete error handling
  - *Superseded by*: Enhanced dashboard implementations in active codebase

- **`test_dashboard.py`** - Test dashboard file
  - *Reason*: Development artifact, not needed for production

- **`test_streamlit.py`** - Streamlit test implementation
  - *Reason*: Experimental file, project moved to other dashboard solutions

### Pipeline (`pipeline/`)
- **`etl_pipeline.py`** - Original ETL pipeline implementation
  - *Issues*: Relied on mock data, limited transformation capabilities
  - *Superseded by*: `smart_etl_pipeline.py` with real Oracle connectivity

- **`etl_pipeline_module/`** - ETL pipeline module directory
  - *Reason*: Module structure that was replaced by simplified single-file approach

### Notebooks (`notebooks/`)
- **`hungerhub_sample_data_analysis_backup.ipynb`** - Backup of sample analysis notebook
  - *Reason*: Backup file no longer needed

- **`hungerhub_sample_data_analysis.ipynb`** - Original sample data analysis
  - *Issues*: Based on small sample sizes, superseded by full data analysis
  - *Superseded by*: `hungerhub_full_data_analysis.ipynb` and `hungerhub_optimized_extraction.ipynb`

- **`hungerhub_comprehensive_analysis.ipynb.backup`** - Backup of comprehensive analysis
  - *Reason*: Backup file preserved for recovery if needed

### Scripts (`scripts/`)
- **`simple_*.py`** - Simple test scripts for Oracle connectivity
  - *Reason*: Development utilities no longer needed in main notebooks directory

- **`sql_diagnostics*.py`** - SQL diagnostic and troubleshooting utilities
  - *Reason*: Debug scripts, useful for reference but not part of main workflow

- **`recovery_extraction.py`** - Data recovery utility script
  - *Reason*: Emergency utility, preserved but not part of standard pipeline

### Launch Scripts (`scripts/launch_scripts/`)
- **`launch_dash_dashboard.sh`** - Comprehensive Dash launcher with deprecated references
  - *Issues*: References `real_data_extractor.py` and incorrect dashboard path
  - *Good features*: Virtual environment management, logging, error handling

- **`launch_dashboard.sh`** - Basic Streamlit launcher with deprecated references
  - *Issues*: References `test_dashboard.py` and incorrect dashboard path

- **`launch_dash_dashboard.sh.backup`** - Backup of deprecated launch script

## 🔄 Migration Notes

### What to Use Instead:

1. **For Data Extraction**: Use `src/data_extraction/full_data_extractor.py`
2. **For Data Unification**: Use `src/data_extraction/create_unified_real_data.py` 
3. **For ETL Pipeline**: Use `src/smart_etl_pipeline.py`
4. **For Dashboards**: Use implementations in `src/dashboard/dash/` or `src/dashboard/streamlit/`
5. **For Analysis**: Use `notebooks/hungerhub_full_data_analysis.ipynb` or `notebooks/hungerhub_optimized_extraction.ipynb`

### Recovery Instructions:
If you need to restore any of these files to active development:

```bash
# Example: Restore the original data extractor for reference
cp src/deprecated/data_extraction/real_data_extractor.py src/data_extraction/
```
